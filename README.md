# Shruti Shukla Portfolio

A public portfolio website for **Shruti Shukla**, a strategic communication specialist working across government campaigns, healthcare communication, tourism storytelling, brand narrative, SEO content architecture, and conversion-oriented product communication.

## Live Website

https://designmavensolution.github.io/shurti-shukla-portfolio/

## About

Shruti's work sits at the intersection of research-led writing, public-facing communication, and brand strategy. The portfolio presents selected work across:

- Government and institutional communication
- Healthcare and medical content strategy
- Tourism and destination storytelling
- Website copy and brand voice development
- Product and eCommerce communication
- Fashion, lifestyle, education, and niche sector writing

## Website

This is a static HTML/CSS/JavaScript site designed for GitHub Pages. It includes:

- A full-screen editorial hero section
- Portfolio case studies and sector-based work sections
- Scroll-based stacked section transitions
- Responsive layout for desktop and mobile
- Public reference links for selected published work

## Project Files

- `index.html` - GitHub Pages entry point
- `portfolio-work.html` - Source portfolio page
- `Shruti Shukla.png` - Portrait image used in the hero section
- `.nojekyll` - Keeps GitHub Pages from applying Jekyll processing
